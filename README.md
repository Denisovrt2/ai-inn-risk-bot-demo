# AI Company Risk Analysis System

## Что это

Система анализа компаний по ИНН, которая собирает данные из разных источников и формирует понятный риск-профиль для принятия решений.

## Задача

В реальности данные о компании разрознены:
- государственные реестры
- судебные данные
- корпоративные изменения
- финансовые и операционные сигналы

Задача системы — не просто собрать данные, а превратить их в понятный сигнал:
стоит работать с компанией или нет.

## Как работает

Pipeline:
1. Сбор данных
2. Очистка и нормализация
3. Формирование признаков
4. Скоринг
5. Генерация AI-отчёта

## Особенности

- Фокус не на генерации текста, а на принятии решения
- Explainable логика: понятно, почему система дала такой вывод
- Работа с неидеальными и разрозненными данными
- Комбинация structured data + AI-слоя

## Пример

Input:
INN: sample

Output:
- уровень риска
- ключевые факторы
- краткий AI-отчёт

## Стек

Python, FastAPI, API, feature engineering, scoring, LLM, RAG

## Статус
